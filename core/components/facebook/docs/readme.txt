## Facebook
Een simpele snippet om die Facebook berichten kan exporteren. Dit kan in JSON en in XML.
Maak een resource aan en laat de snippet de juiste data tonen in JSON of XML aan de hand van de onderstaande snippet voorbeelden. Deze resource kun je vervolgens weer aanroepen via Ajax of RSS feed functie in de narrowcasting (zie voorbeeld Sterc intranet).

XML voorbeeld: [[!Facebook? &tpl=`facebookItemTpl`&tplWrapper=`facebookWrapperTpl`]]
JSON voorbeeld: [[!Facebook? &toJson=`1`]]
